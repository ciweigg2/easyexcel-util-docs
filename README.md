## easyexcel

### easyexcel封装方法

主要封装日常开发常用的方法：

* 同步读（读入数据无模板）
* 同步读（按模板读入数据）
* 异步读（读入数据无模板）
* 异步读（按模板读入数据）
* 写文件（元数据无模板）
* 写文件（通过注解定义模板对象元数据）
* 写文件（写一个excel的多个sheet页）
* 写文件（web导出并下载文件）
* 写文件（通过注解配置支持导出文件单元格下拉菜单）
* 写文件（分页查询并写入文件）
* 获取表头信息
* 批量读取数据（实现BaseProcessor接口批量存储）
* 批量读取数据并校验数据（实现BaseProcessor接口批量存储）
